# Virtual Midi Janko keyboard

![](http://i.imgur.com/amtjcq5.gif)

VMJK is a virtual midi keyboard using janko key layout, written in Rust.

## Build

To build VMJK you need CSFML and rust.

```
% cargo build --release
```
